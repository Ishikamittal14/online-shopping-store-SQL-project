# Online Shopping Store SQL Project

## ✅ Overview
This project models an **Online Shopping Store Database**.  
It includes tables to manage:
- Customers
- Products
- Orders
- Cart Items
- Order Items
- Payments  

The project demonstrates practical SQL concepts:
- Table creation with primary & foreign keys  
- Data insertion  
- JOIN operations  
- Aggregation using GROUP BY  
- Maintaining referential integrity

## ✅ How to Use

1. Run `create_tables.sql` to create the database schema.
2. Run `insert_data.sql` to populate sample data in all tables.
3. Run `sample_queries.sql` to execute example queries for:
    - Retrieving products in orders
    - Calculating total payments per customer
    - Viewing customer cart items
